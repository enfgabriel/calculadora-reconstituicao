Calculadora de reconstituição de medicamentos + dose para seringas U-100. Responsiva, com modo escuro e cálculos precisos. Desenvolvida em HTML/CSS/JS puro.
Personalização
Todas as cores e estilos estão definidos com variáveis CSS (:root e body.dark).
Para alterar o esquema de cores, edite as variáveis no início do arquivo <style>.


HTML5 – estrutura semântica
CSS3 – flexbox, grid, variáveis, responsividade e animações suaves
JavaScript (ES6) – lógica de cálculo, interatividade, persistência do tema
Nenhuma dependência externa ou framework – tudo em um único arquivo.


Sinta-se livre para usar, modificar e distribuir, mantendo os créditos.

Desenvolvido para profissionais da saúde e estudantes.
Se encontrar algum problema ou tiver sugestão, abra uma issue no repositório.
